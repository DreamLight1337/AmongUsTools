# Among Us Tools (AUTools)
AUTools 是一个专为 Among Us 私服安装设计的工具箱，支持一键添加多个私服到游戏中，同时还提供了一些有趣的附加功能。
目前支持清风服、帆船服、核电站服、碧水港服和Niko服。
